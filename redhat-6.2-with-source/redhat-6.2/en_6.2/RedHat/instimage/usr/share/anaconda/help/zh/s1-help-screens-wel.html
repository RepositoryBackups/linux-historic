欢迎使用 Red Hat Linux

欢迎！Red Hat Linux Installation Guide（可从 Red Hat, Inc. 获得）中详细说明了整个安装过程。开始安装之前，请仔细阅读整本手册。

http://www.redhat.com 中联机提供了此手册的 HTML 和 PostScript 副本。Red Hat Linux CD 的 doc 目录中也提供了此手册的 HTML 副本。

如果您已购买一份正式版 Red Hat Linux，请务必通过我们的 Web 站点 (http://www.redhat.com/now) 注册您的产品。

整个安装过程中，您均可以使用鼠标选择不同的安装选项，也可以使用 Tab 键和 Enter 键以控制安装过程。

您可以使用“下一步”和“上一步” 按钮以浏览这些屏幕。单击 “下一步”按钮可以保存信息并进入下一屏幕；单击“上一步”按钮可以返回上一屏幕，并且不保存任何信息。

要最小化此帮助屏幕，请单击“隐藏帮助” 按钮。

如果您由于某种原因而需要取消此安装，则可以使用“重置”按钮或 Ctrl-Alt-Del 组合键，以安全地重新引导系统。在显示 X 配置屏幕之前，您可以随时取消安装。一旦已配置 X，将开始安装软件包，并且安装程序会将数据写入您的硬盘驱动器。
