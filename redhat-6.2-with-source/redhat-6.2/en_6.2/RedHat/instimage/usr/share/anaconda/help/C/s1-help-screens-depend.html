Unresolved Dependencies

Many software packages depend on other packages or libraries in order to work correctly. To make sure your system has all the packages it needs in order to work, Red Hat Linux checks these package dependencies each time you install or remove a package. If one package requires another package that has not been installed, unresolved dependencies exist.

One or more packages you've selected have unresolved dependencies. You can resolve this by selecting Install Packages to Satisfy Dependencies.
