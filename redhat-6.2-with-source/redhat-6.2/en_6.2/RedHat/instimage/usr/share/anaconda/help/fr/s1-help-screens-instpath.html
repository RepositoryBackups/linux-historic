Options d'installation

Comment souhaitez-vous installer Red Hat Linux ?

Une installation complète détruira toutes les informations précédemment sauvegardées sur les partitions sélectionnées.

Une mise à jour permet de conserver les données système Red Hat Linux existantes.

Si vous souhaitez procéder à une installation complète, vous devez choisir la classe (ou le type) de l'installation. Les options disponibles sont : Station de travail GNOME, Station de travail KDE, Serveur ou Personnalisée.

Si vous avez l'habitude d'utiliser l'outil de partitionnement fdisk , vous pouvez le sélectionner à ce niveau. Si vous ne choisissez pas d'utiliser fdisk, le programme d'installation utilisera l'outil de partitionnement par défaut, Disk Druid. Nous vous recommandons d'utiliser Disk Druid si vous ne connaissez pas fdisk.

Si vous ne savez pas quelle classe d'installation utiliser, lisez attentivement les informations suivantes.

Remarque : outre les méthodes d'installation mentionnées ci-dessus, l'installation de Red Hat Linux sur une partition FAT (DOS/Windows) existante est également à votre disposition. Cette installation "sans partition" est sélectionnée dès que vous affectez le label / à une partition DOS dans l'écran de partitionnement de Disk Druid. Elle remplace la suppression automatique de la partition utilisée dans les installations de la classe Station de travail.

Les installations de la classe Station de travail permettent d'installer le système X Window et le gestionnaire de bureau de votre choix. Une installation de cette classe supprime les partitions liées à Linux sur tous les disques durs installés (et utilise tout l'espace disque non partitionné). Les partitions non liées à Linux ne sont pas modifiées, et vous pourrez démarrer d'autres systèmes d'exploitation que Red Hat Linux une fois l'installation terminée.

Si vous souhaitez que votre système fonctionne comme un serveur Linux et que vous ne voulez pas effectuer une configuration trop complexe, ni installer le système X Window, une installation de la classe Serveur est plus appropriée. Elle supprime TOUTES les partitions existantes sur TOUS les disques durs installés ; vous ne devez la sélectionner que si vous êtes sûr de ne vouloir conserver aucune donnée existante ! Cela inclut TOUTES les partitions utilisées par d'autres systèmes d'exploitation. Le contenu de TOUS les disques va être effacé.

Seule l'installation personnalisée vous offre une flexibilité totale. Une installation personnalisée vous permet de choisir l'espace disque que vous souhaitez partitionner. Vous avez un contrôle total sur les paquetages qui seront installés sur votre système. Par ailleurs, vous pouvez également déterminer si LILO doit prendre en charge le démarrage de votre système. Vous ne devez choisir ce type d'installation que si vous connaissez déjà bien Linux.

Pour plus d'informations sur les différentes classes d'installation (Station de travail, Serveur et Personnalisée), reportez-vous au Red Hat Linux Installation Guide.
