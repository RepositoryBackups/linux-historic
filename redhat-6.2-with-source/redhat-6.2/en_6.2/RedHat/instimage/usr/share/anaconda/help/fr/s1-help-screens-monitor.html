Configuration X

Le programme d'installation va maintenant tester votre écran pour déterminer les paramètres d'affichage les plus adaptés à votre ordinateur. Si le programme ne parvient pas à déterminer l'écran que vous utilisez, choisissez dans la liste l'écran le plus proche de celui que vous utilisez.

Vous pouvez également entrer les fréquences horizontale et verticale de votre écran. Vous pouvez trouver ces valeurs dans la documentation de votre matériel. Veillez à entrer les valeurs appropriées ; si vous entrez des valeurs dépassant les capacités de votre matériel, vous risquez d'endommager votre écran. N'entrez des valeurs dans ces champs que si les valeurs de votre manuel ne correspondent pas à celles de la liste des écrans et si vous êtes certain que les valeurs figurant dans votre documentation sont correctes.
