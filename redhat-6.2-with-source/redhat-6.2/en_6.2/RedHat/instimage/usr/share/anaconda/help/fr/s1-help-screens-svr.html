Partitionnement automatique

Etes-vous sûr ?

Si vous exécutez une installation Serveur, vous perdrez toutes vos données.

Cette classe d'installation supprimera TOUTES les données de TOUTES les partitions sur TOUS vos disques.

Si vous utilisez un autre système d'exploitation que vous voulez conserver, ne sélectionnez pas cette classe.

Pour conserver des données ou un autre système d'exploitation, vous devez effectuer la partition manuellement ou sélectionner l'installation personnalisée.

Utilisez le bouton Précédent pour sélectionner l'installation personnalisée ou Suivant pour continuer l'installation Serveur.
