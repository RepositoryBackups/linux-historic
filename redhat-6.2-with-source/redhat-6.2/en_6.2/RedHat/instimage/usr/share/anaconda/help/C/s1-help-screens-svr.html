Automatic Partitioning

Are you sure?

You will lose data by performing a server installation.

This class of installation will remove ALL data on ALL partitions of ALL disk drives.

If you have another OS that you want to keep, do not choose this installation class.

If you want to preserve data or another OS, you should choose to partition manually or choose a custom-class installation.

Use the Back button to choose a custom-class installation, or choose Next if you want to proceed with a server-class installation.
