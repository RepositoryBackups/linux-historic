Configuration de la souris

Quel type de souris utilisez-vous ?

Utilisez-vous une souris PS/2, bus ou série ? (Conseil : si le connecteur auquel votre souris est branchée est rond, vous utilisez une souris PS/2 ou bus ; s'il est rectangulaire, il s'agit d'une souris série.)

Essayez de trouver votre modèle dans la liste. Si vous ne trouvez pas le modèle exact, vous pouvez choisir un modèle compatible avec le vôtre. Sinon, sélectionnez le type de souris générique approprié.

Si vous utilisez une souris série, choisissez le port auquel elle est connectée dans la boîte de dialogue suivante.

Le système X Window est conçu pour utiliser une souris à trois boutons. Si vous utilisez une souris à deux boutons, Red Hat Linux peut émuler le bouton du milieu.

Si vous utilisez une souris à deux boutons, sélectionnez l'option Emulation de souris à trois boutons. Après l'installation, vous pourrez cliquer sur les deux boutons en même temps pour simuler le bouton du milieu.
