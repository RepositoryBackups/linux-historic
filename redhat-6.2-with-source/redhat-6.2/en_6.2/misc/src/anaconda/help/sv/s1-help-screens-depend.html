Olösta paketberoenden

Många programpaket behöver andra paket eller bibliotek för att fungera korrekt. För att säkerställa att ditt system har alla de paket det behöver för att fungera kollar Red Hat Linux dessa paketberoenden varje gång du installerar eller tar bort ett paket. Om ett paket behöver ett annat paket som inte är installerat finns ett paketberoende.

Ett eller flera paket som du har valt har olösta beroenden. Du kan lösa detta genom att välja Installera paket för att tillfredsställa beroenden.
