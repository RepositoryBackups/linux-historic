Разделы

Куда вы хотите установить Red Hat Linux?

Необходимо отметить: если вы выполняете Установку без переразбиения, вы должны пометить один из существующих разделов DOS/Windows как корневой раздел. Выделите выбранный для установки раздел FAT и нажмите кнопку Edit для назначения точки монтирования и в качастве точки монтирования укажите / (root). Когда вы подтвердите выбор, вам необходимо будет указать размеры корневой файловой системы и пространства свопинга.

За более подробными инструкциями необходимо обратиться к разделу Установка без переразбиения в Red Hat Linux Installation Guide.

Если вы не уверены, как лучше разбить диск на разделы, обратитесь к соответствующему разделу Red Hat Linux Installation Guide, или Red Hat Linux Reference Guide.

Если вы только что завершили разбиение диска программой fdisk, вам необходимо назначить точки монтирования для созданных вами разделов. Выберите раздел и нажмите кнопку Edit для назначения точки монтирования.

Если вы используете приложение Disk Druid, вы должны назначить точки монтирования для одного или нескольких разделов. Здесь же можно создавать или удалять разделы.

В верхней части экрана показано текущее состояние таблицы разделов. Каждый раздел имеет пять полей:

  • Mount Point: Это поле показывает, как будет монтироваться раздел при загрузке установленного Red Hat Linux. Если раздел существует но точка монтирования не назначена, вам необходимо назначить ее, вызвав соответствующий диалог двойным щелчком по строке раздела или нажатием кнопки Edit.

  • Device: Это поле показывает имя устройства, на котором находится раздел (например, hda2 означает второй раздел на первом жестком диске).

  • Requested: Поле Requested показывает, какой размер был назначен для этого раздела. Если вы хотите изменить размер раздела, вам необходимо удалить этот раздел и создать его снова, используя кнопку Add.

  • Actual: Это поле показывает действительный размер раздела.

  • Type: Это поле показывает тип раздела (например, Linux Native or DOS).

При просмотре списка разделов вы можете увидеть надпись Unallocated Requested Partition красного цвета около одного или более разделов. Это сообщение может возникнуть, если разделу был назначен размер, превышающий свободное место на диске.

Для исправления этой ошибки вы можете либо перенести раздел на другой диск, имеющий больше свободного места, либо изменить рамер раздела, либо удалить его целиком. Вызовите диалог изменения настроек раздела двойным щелчком по его строке либо нажмите кнопку Edit.

Разбиение дисков вашей системы

Центральный ряд кнопок управляет функциями утилиты разбиения дисков. Здесь вы можете добавлять, редактировать и удалять разделы. В дополнение, имеются кнопки для сохранения сделанных изменений, или отмены и выхода из утилиты разбиения.

  • Add: Используйте эту кнопку для создания нового раздела. При этом появляется окно диалога с полями, которые необходимо заполнить (такие как точка монтирования, размер и тип). type).

  • Edit: Используйте эту кнопку для изменения точки монтирования текущего раздела.

  • Delete: Используйте эту кнопку для удаления выбранного раздела. Должен появиться диалог, запрашивающий подтверждение удаления.

  • Reset: Используйте эту кнопку для отмены всех сделанных вами изменений.

  • Make RAID Device: Используйте кнопку Make RAID Device ТОЛЬКО если у вас есть опыт использования RAID. Чтобы больше узнать о RAID, обратитесь к Red Hat Linux Reference Guide.

Drive Summaries

Эта секция показывает основные сведения о дисках, установленных в вашем компьютере. Каждая строка содержит следующие поля:

  • Drive: имя дискового устройства (например hda или hdb).

  • Geom [C/H/S]: Геометрия жесткого диска показывает количество цилиндров, головок и секторов.

  • Total: Это поле показывает полный объем жесткого диска

  • Free: Здесь можно увидеть, сколько места на диске все еще доступно.

  • Used: Это поле показывает, какой объем жесткого диска уже распределен между разделами, в мегабайтах и процентах.
