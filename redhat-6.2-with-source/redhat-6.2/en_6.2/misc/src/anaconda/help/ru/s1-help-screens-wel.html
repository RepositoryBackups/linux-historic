Добро пожаловать в Red Hat Linux

Добро пожаловать! Этот процесс установки подробно описан в Red Hat Linux Installation Guide которое распространяется Red Hat, Inc.. Пожалуйста, внимательно прочитайте это руководство перед началом процесса установки.

Копии руководства в формате HTML и Postscript доступны по адресу http://www.redhat.com. Копия руководства в формате HTML находится также на Red Hat Linux CD.

New Feature: Red Hat Linux 6.2 имеет новый метод установки, который не требует дополнительного разбиения диска ("partitionless"). Если в вашей системе есть раздел FAT (DOS/Windows), на котором достаточно свободного места, вы можете установить Red Hat Linux без переразбиения жесткого диска. Этот метод наиболее подходит тем, кто хочет ознакомится с Red Hat Linux, не внося серьезных изменений в конфигурацию своей системы.

Если вы приобрели официальный комплект Red Hat Linux, не забудьте зарегистрировать вашу копию на нашем Web-сайте (www.redhat.com/now).

В процессе инсталляции вы сможете выбирать различные опции при помощи мыши. Для навигации в процессе установки можно использовать также клавиши Tab and Enter.

Используйте кнопку Далее для сохранения информации и перехода к следующему экрану. Кнопка Назад возвращает к предыдущему этапу без сохранения введенной информации.

Для минимизации этого экрана помощи нажмите кнопку Скрыть помощь.

Вы можете прервать инсталляцию в любое время, вплоть до экрана Подготовка к установке. После нажатия кнопки Далее на экране Подготовка к установке будет начата установка пакетов на жесткий диск. Для прерывания установки перед этим экраном вы можете безопасно перезагрузить вашу систему (используйте кнопку reset, или Ctrl-Alt-Del).
