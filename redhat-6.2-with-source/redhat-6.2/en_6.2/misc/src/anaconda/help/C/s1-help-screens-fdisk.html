fdisk

Select which drive you want to partition. When you click on the drive, you will be presented with the fdisk partitioning screen. You may then use fdisk to create, delete, or modify partitions on the selected hard drive.

If you decide that you don't want to use fdisk, click Back to return to the previous screen, deselect Use fdisk, and click Next to continue.

Once you have partitioned the drive using fdisk, you will return to this screen. If you have another hard drive that you would like to partition using fdisk, select it. Otherwise, click Next to assign mount points to your partitions using Disk Druid.
