Red Hat Linux vous souhaite la bienvenue !

Bienvenue ! Ce processus d'installation est présenté en détails dans le Red Hat Linux Installation Guide disponible auprès de Red Hat, Inc.. Lisez la totalité du guide avant de commencer l'installation.

Des versions HTML et PostScript du guide sont disponibles en ligne sur le site http://www.redhat.com. Il existe également une version HTML sur le CD-ROM Red Hat Linux.

Nouvelle fonctionnalité : Red Hat Linux 6.2 bénéficie d'une nouvelle méthode d'installation appelée installation "sans partition". Si votre système comporte une partition FAT (DOS/Windows) avec un espace disque disponible suffisant, vous pouvez installer Red Hat Linux sans repartitionner votre disque dur. Cette option présente un intérêt particulier pour les utilisateurs souhaitant essayer Red Hat Linux sans pour autant interrompre pendant une longue durée le fonctionnement du système déjà installé.

Si vous avez acheté la boîte Red Hat Linux Officiel, n'oubliez pas d'enregistrer votre produit sur notre site Web (www.redhat.com/now).

Tout au long de cette installation, vous pourrez utiliser votre souris pour sélectionner différentes options. Vous pouvez naviguer dans l'installation à l'aide des touches Tab et Entrée.

Utilisez les boutons Suivant et Précédent pour naviguer parmi les écrans. Cliquez sur Suivant pour sauvegarder les informations et afficher l'écran suivant ; cliquez sur Précédent pour revenir à l'écran précédent sans sauvegarder les informations.

Pour réduire cet écran d'aide, cliquez sur le bouton Masquer l'aide.

Vous pouvez annuler cette installation à tout moment jusqu'à l'écran Préparation de l'installation. Une fois que vous avez cliqué sur Suivant dans l'écran Préparation de l'installation, le système lance l'installation des paquetages et commence à enregistrer des données sur votre disque dur. Pour annuler l'installation avant cet écran, vous pouvez redémarrer votre ordinateur en toute sécurité (utilisez le bouton reset, ou Ctrl-Alt-Suppr).
