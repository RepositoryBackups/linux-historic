Installation des paquetages

Nous avons rassemblé toutes les informations nécessaires à l'installation de Red Hat Linux sur votre système. L'installation peut prendre un certain temps, selon le nombre de paquetages que vous installez. Si vous disposez d'une boîte Red Hat Linux Officiel, vous pouvez en profiter pour commencer à consulter le Red Hat Linux Getting Started Guide ou le Red Hat Linux Reference Guide.
